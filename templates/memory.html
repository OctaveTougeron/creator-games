Comment jouer à Memory

Memory est un jeu de cartes où le but est de retrouver les paires identiques parmi des cartes faces cachées. Ce jeu fait appel à la mémoire et à la concentration.

But du jeu :

Retrouver le plus de paires de cartes identiques.

Préparation :

Mélangez toutes les cartes et disposez-les face cachée en rangées sur la table. Assurez-vous que toutes les cartes sont bien mélangées.

Règles :

  • Le premier joueur retourne deux cartes de son choix.
  • Si les cartes sont identiques, il les remporte et joue à nouveau.
  • Si les cartes sont différentes, elles sont retournées face cachée, et c'est au tour du joueur suivant.
  • Le jeu continue jusqu'à ce que toutes les paires aient été trouvées.

Gagnant :

Le joueur ayant trouvé le plus de paires remporte la partie.
